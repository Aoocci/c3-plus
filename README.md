# Aoocci C3 Plus — Firmware & Docs

5" wireless Apple CarPlay / Android Auto smart monitor.

- **Official site:** https://aoocci.com
- **Update guide, tools (PhoenixCard) & support:** https://aoocci.com/pages/motorcycle-support

## Firmware
Download the latest firmware from the **[Releases](../../releases)** tab — each release lists version, changelog and SHA256.

> ⚠️ Flash only the firmware for **your exact model**, per the official support-page instructions. Updating the wrong file can brick the device.
